# Dynamic Business Dashboard (Excel)

## Project Overview
Created a Dynamic Business Dashboard in Excel to analyze sales performance, profitability, and salesperson efficiency using interactive visuals.

## Business Objectives
- Track total sales, boxes sold, and profit percentage
- Identify top-performing products
- Analyze salesperson performance and growth
- Enable dynamic filtering by product and category

## Key Insights
- Top 5 products contribute a major share of revenue
- Clear comparison of salesperson performance
- Monthly trends help track sales, boxes, and profit
- Interactive filters allow dynamic business analysis

## Dashboard Features
- KPI cards for Sales, Boxes, and Profit %
- Product-wise and category-wise analysis
- Salesperson performance table
- Salesperson growth chart
- Interactive slicers for Product and Category

## Skills & Tools Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Advanced Excel formulas
- Slicers
- Dashboard design
